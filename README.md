# DebtTracker

**Debt Management System**

## Overview

DebtTracker is a basic Django-based debt management system designed with structured user workflows for tracking, requesting, and settling repayments.

## Key Features

- **Structured Workflows:** Built-in processes for tracking, requesting, and settling debt repayments.
- **Secure Authentication:** Implemented secure user authentication and session handling using Django’s built-in auth system, including protected routes.
- **Relational Data Management:** Structured backend logic utilizing Django models and ORM to efficiently manage relational data between users and financial transactions.

## Technologies Used

- **Backend framework**: Django
- **Database**: SQLite
